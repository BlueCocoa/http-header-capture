# HTTP Header Capture
A simple HTTP header capture program with syntax highlight feature (Solarized style)

Tested on macOS

### Compile
```$ make && sudo make install```


### Dependencies 
- libpcap
- [xterm256](https://github.com/BlueCocoa/xterm256)
